# History and Evolution of Physical AI

## Learning Objectives
- Trace the historical development of robotics and AI
- Identify key milestones in humanoid robot development
- Understand how AI and robotics converged

## Timeline of Key Milestones

| Year | Event | Significance |
|------|-------|--------------|
| 1956 | AI term coined | Dartmouth Conference |
| 1961 | Unimate | First industrial robot |
| 1966 | Shakey | First reasoning robot |
| 1996 | Honda P2 | Modern humanoid |
| 2000 | ASIMO | Consumer awareness |
| 2013 | Atlas | Dynamic locomotion |
| 2023 | Humanoid race | Multiple companies |

## The Modern Era

Deep learning (2012+) transformed robot perception. We are now in a humanoid race with Tesla, Figure, and others competing.

## Key Takeaways

:::tip Summary
- Robotics evolved from simple automatons to sophisticated humanoids
- Deep learning dramatically improved robot capabilities
- Multiple companies now pursue humanoid robots
:::
