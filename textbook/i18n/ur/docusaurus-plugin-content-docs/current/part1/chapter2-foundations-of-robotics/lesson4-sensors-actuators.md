# Sensors and Actuators

![Sensor Types](/img/diagrams/sensor-types.svg)

## Sensors
- Vision: Cameras, depth sensors
- Proprioception: Encoders, IMUs
- Touch: Force/torque sensors

## Actuators
- Electric motors: DC, servo, stepper
- Hydraulics: High force
- Pneumatics: Soft robotics

## Key Takeaways
:::tip Summary
- Sensors provide perception
- Actuators enable action
:::
