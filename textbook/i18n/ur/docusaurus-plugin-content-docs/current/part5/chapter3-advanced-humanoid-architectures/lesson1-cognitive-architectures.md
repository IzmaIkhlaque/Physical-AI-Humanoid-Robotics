# ہیومنائڈز کے لیے ادراکی فن تعمیرات

## خلاصہ (حصہ 5: سماجی انضمام اور جدید تصورات)

یہ حصہ ہیومنائڈ روبوٹس کو ہماری روزمرہ کی زندگی میں ضم کرنے کے عملی اور اخلاقی مضمرات کا جائزہ لیتا ہے، ان کے ممکنہ کرداروں اور ان سماجی تبدیلیوں کو تلاش کرتا ہے جو وہ لا سکتے ہیں۔

ہیومنائڈ روبوٹس کو انسانی سطح کی ذہانت اور موافقت حاصل کرنے کے لیے صرف جدید سینسرز، چست جسموں اور نفیس مشین لرننگ الگورتھمز سے زیادہ کی ضرورت ہے۔ انہیں ایک ایسے جامع فریم ورک کی ضرورت ہے جو ان اجزاء کو ایک مربوط، ذہین نظام میں ضم کرے جو انسانی ذہن کی طرح استدلال، منصوبہ بندی، سیکھنے اور تعامل کرنے کی صلاحیت رکھتا ہو۔ یہ **ادراکی فن تعمیرات** کا دائرہ ہے: حسابی ماڈلز جو انسانی ادراکی صلاحیتوں کی وسیع رینج کو حاصل کرنے کے لیے ڈیزائن کیے گئے ہیں، جو ہیومنائڈز کو ذہانت سے سوچنے، سمجھنے اور عمل کرنے کے قابل بناتے ہیں۔

### 1. ادراکی فن تعمیرات کیا ہیں؟

ایک ادراکی فن تعمیر انسانی ذہن (یا ایک مصنوعی ذہن) کی ساخت کا ایک نظریہ ہے جو بنیادی، مقررہ حسابی اجزاء اور ان کی تنظیم کو بیان کرتا ہے، جس کے ذریعے تمام ادراک ہوتا ہے۔ یہ اس بات کی وضاحت کرتا ہے کہ ادراک، یادداشت، استدلال، سیکھنے اور عمل کو کس طرح مربوط کیا جاتا ہے اور ذہین رویے کو پیدا کرنے کے لیے کس طرح تعامل کرتے ہیں۔ ہیومنائڈز کے لیے، ایک ادراکی فن تعمیر وہ "دماغ" فراہم کرتا ہے جو روبوٹ کے ادراک-عمل کے چکر کو منظم کرتا ہے۔

#### اہم افعال:

*   **ادراک:** روبوٹ کی آنکھوں، کانوں، ٹچ سینسرز سے حسی ڈیٹا کی تشریح کرنا۔
*   **یادداشت:** ماضی کے تجربات، حقائق، مہارتوں (طریقہ کار کی یادداشت)، اور تصوراتی علم کو ذخیرہ کرنا اور بازیافت کرنا۔
*   **استدلال اور مسئلہ حل کرنا:** نئے حالات سے نمٹنے اور اہداف حاصل کرنے کے لیے منطق، استدلال اور ہورسٹکس کا استعمال کرنا۔
*   **سیکھنا:** رویوں کو اپنانا، نئی مہارتیں حاصل کرنا، اور علم کو اپ ڈیٹ کرنا۔
*   **عمل کا انتخاب اور عملدرآمد:** یہ فیصلہ کرنا کہ کیا کرنا ہے اور جسم کو اسے کرنے کے لیے کیسے کنٹرول کرنا ہے۔
*   **توجہ:** متعلقہ معلومات پر ادراکی وسائل کو مرکوز کرنا۔

### 2. ادراکی فن تعمیرات کی مثالیں

کئی نمایاں ادراکی فن تعمیرات تیار کی گئی ہیں، ہر ایک مختلف زور کے ساتھ:

#### الف. SOAR (حالت، آپریٹر، اور نتیجہ)

*   **توجہ:** مسئلہ حل کرنا اور تجربے سے سیکھنا۔
*   **طریقہ کار:** مسئلہ کی جگہوں، آپریٹرز، اور ایک عالمگیر ذیلی ہدف کے طریقہ کار پر مبنی۔ تمام علم قواعد کے طور پر پیش کیا جاتا ہے، اور سیکھنا "چنکنگ" (مسئلہ حل کرنے کے نتائج کو نئے قواعد میں مرتب کرنا) کے ذریعے ہوتا ہے۔
*   **ہیومنائڈز پر اطلاق:** ہیومنائڈز کو کاموں کے بارے میں استدلال کرنے، اعمال کی ترتیب کی منصوبہ بندی کرنے، اور اپنی کامیابیوں اور ناکامیوں سے سیکھنے کے قابل بناتا ہے۔

#### ب. ACT-R (سوچ کا انطباقی کنٹرول—منطقی)

*   **توجہ:** انسانی ادراک کی ماڈلنگ، خاص طور پر یادداشت اور مہارت کا حصول۔
*   **طریقہ کار:** علامتی اور ذیلی علامتی پروسیسنگ کو ضم کرتا ہے، جس میں اعلانیہ یادداشت (حقائق)، طریقہ کار کی یادداشت (مہارتیں)، اور ادراکی-موٹر انٹرفیس کے لیے الگ ماڈیولز ہوتے ہیں۔ سیکھنا پروڈکشن رول کی تالیف اور پیرامیٹر ایڈجسٹمنٹ کے ذریعے ہوتا ہے۔
*   **ہیومنائڈز پر اطلاق:** ہیومنائڈز کو انسانی جیسی موٹر مہارتیں اور اعلانیہ علم حاصل کرنے کے لیے ایک فریم ورک فراہم کر سکتا ہے، جو زیادہ قدرتی تعامل کی حمایت کرتا ہے۔

#### ج. CLARION (انطباقی اصول انڈکشن آن لائن کے ساتھ کنکشنسٹ لرننگ)

*   **توجہ:** مضمر (لا شعوری) اور واضح (شعوری) علم کو یکجا کرنا، اور وسیع رینج کے کاموں کو سیکھنا۔
*   **طریقہ کار:** ایک دوہری نمائندگی کا نظام جس میں مضمر علم کے لیے ایک باٹم-اپ (کنکشنسٹ) پرت اور واضح علم کے لیے ایک ٹاپ-ڈاؤن (علامتی) پرت ہوتی ہے۔
*   **ہیومنائڈز پر اطلاق:** سیکھی ہوئی نچلی سطح کی موٹر مہارتوں کو اعلیٰ سطح کی علامتی منصوبہ بندی اور استدلال کے ساتھ ضم کرنے کا ایک طریقہ پیش کرتا ہے، جو زیادہ مضبوط اور لچکدار رویے کو ممکن بناتا ہے۔

### 3. ادراکی فن تعمیرات کو ہیومنائڈ ہارڈ ویئر کے ساتھ مربوط کرنا

ہیومنائڈز کے لیے ادراکی فن تعمیرات کی حقیقی طاقت روبوٹ کے فزیکل جسم کے ساتھ ان کے گہرے انضمام سے آتی ہے:

*   **مجسم ادراک (Embodied Cognition):** یہ خیال کہ روبوٹ کا ادراک اس کے فزیکل جسم اور فزیکل دنیا کے ساتھ اس کے تعاملات سے گہرا متاثر ہوتا ہے۔ فن تعمیر کو سینسرز اور ایکچویٹرز کے ساتھ مؤثر طریقے سے انٹرفیس کرنا چاہیے۔
*   **ریئل ٹائم پروسیسنگ:** ادراکی افعال کو ایک فزیکل روبوٹ کے اپنے ماحول پر ردعمل ظاہر کرنے کی ٹائمنگ کی رکاوٹوں کے اندر کام کرنا چاہیے۔
*   **ماڈیولر ڈیزائن:** فن تعمیر کو ماڈیولرٹی کی اجازت دینی چاہیے، جو مختلف ادراکی اجزاء (مثلاً، بصری پروسیسنگ، زبان کی تفہیم، موٹر کنٹرول) کو تیار اور مربوط کرنے کے قابل بنائے۔

**کوڈ کا نمونہ (ہیومنائڈ کے لیے تصوراتی ادراکی لوپ):**

```python
import time
import random

class HumanoidCognitiveArchitecture:
    def __init__(self, name="CogBot"):
        self.name = name
        self.perception_buffer = [] # Store recent sensor data
        self.working_memory = {}   # Short-term memory for current context
        self.long_term_memory = {} # Stored facts, skills, experiences
        self.goals = []
        self.current_action = "idle"

    def perceive(self, sensory_input):
        # Simulate processing sensory input
        self.perception_buffer.append(sensory_input)
        if len(self.perception_buffer) > 10: # Keep buffer limited
            self.perception_buffer.pop(0)
        
        # Example: if a human is detected, update working memory
        if "human_detected" in sensory_input:
            self.working_memory["human_present"] = True
